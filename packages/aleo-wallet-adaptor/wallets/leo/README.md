# @provablehq/aleo-wallet-adaptor-leo

Adapter for integrating the Leo wallet with the Aleo wallet adaptor stack.

## Install

```bash
pnpm add @provablehq/aleo-wallet-adaptor-leo
```

Use it with the React provider:

```tsx
import { LeoWalletAdapter } from '@provablehq/aleo-wallet-adaptor-leo';

const wallets = [new LeoWalletAdapter()];
```

Demo application: https://aleo-dev-toolkit-react-app.vercel.app/
