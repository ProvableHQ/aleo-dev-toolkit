# @provablehq/aleo-wallet-adaptor-soter

Adapter for the Soter wallet within the Aleo wallet adaptor suite.

## Install

```bash
pnpm add @provablehq/aleo-wallet-adaptor-soter
```

Usage example:

```tsx
import { SoterWalletAdapter } from '@provablehq/aleo-wallet-adaptor-soter';

const wallets = [new SoterWalletAdapter()];
```

Demo: https://aleo-dev-toolkit-react-app.vercel.app/
