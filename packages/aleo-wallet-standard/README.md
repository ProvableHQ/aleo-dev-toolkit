# @provablehq/aleo-wallet-standard

Type definitions and capability interfaces that define the Aleo wallet adaptor standard.

## Install

```bash
pnpm add @provablehq/aleo-wallet-standard
```

Use the exported enums and interfaces to implement wallet integrations or to build on top of the adaptor core.

Demonstrated in the live example: https://aleo-dev-toolkit-react-app.vercel.app/
